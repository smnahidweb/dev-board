Question-3: What is event delegation in the DOM?

Event delegation is the java script method that maintain event bubbling using by single event listener to a parent element instead of multiple child elements. when When an event occurs, it propagates up the DOM hierarchy, allowing us to capture and handle it using event.target. This method optimizes performance, simplifies code management, and ensures seamless event handling, even for dynamically generated elements

Question-4: What is event bubbling in the DOM?

Event bubbling is java script mechanism where event triggered on a child element first executes its own event handler and then propagates (or "bubbles up") to its parent and ancestor elements. if we have multiple child node, and if i make triggered in the parent node, then all child node will be exccuted. but if i use event.stopPropagation() in the specific child then it will stop and dont upword bubbling. we have another method that is event.ImmidiatedStopPropagation(), then it prevents others listener on the same elements.
